# Tab Organizer Chrome Extension

A simple and useful Chrome extension to organize your browser tabs efficiently.

##  Features

- Group tabs by website (domain)
- Close all tabs of a specific site with one click
- Pin / Unpin tabs easily
- Highlight favorite sites
- Clean and user-friendly interface

---

##  Installation Guide

1. Download the project as ZIP  
2. Extract the ZIP file (**important**)  
3. Open Chrome and go to: chrome://extensions/  
4. Enable **Developer Mode** (top right)  
5. Click **Load Unpacked**  
6. Select the extracted folder (not the ZIP file)  

---

##  Important

-  Do NOT select the ZIP file  
- Always extract it first and then select the folder  

---

##  How to Use

- Click on the extension icon  
- View all open tabs grouped by website  
- Use buttons to:
  - Close all tabs of a site  
  - Pin / Unpin tabs  
- Favorite sites will be highlighted
