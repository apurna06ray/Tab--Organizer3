What this Project Does

This Chrome Extension helps you organize your browser tabs easily. You can group tabs by website, close all tabs of a site, pin/unpin them, and highlight your favorite sites.

Features

  • Group tabs by website (domain)
  • Close all tabs of a site in one click
  • Pin / Unpin tabs quickly
  • Highlight favorite sites

How to Install

  1. Download the ZIP using the button above
  2. Extract the ZIP file (VERY IMPORTANT)
  3. Open Chrome and go to chrome://extensions/
  4. Turn ON Developer Mode
  5. Click Load Unpacked
  6. Select the extracted folder (not ZIP)
❌ Do NOT select the ZIP file
✅ Always extract it and select the folder

How to Extract ZIP File

  1. Go to your Downloads folder
  2. Right-click on the ZIP file
  3. Click "Extract All" or "Extract Here"
  4. A new folder will be created
  5. Use this folder in Chrome

How to Use

  • Click the extension icon in Chrome
  • View all tabs grouped by website
  • Use buttons to close or pin tabs
  • Favorite sites will be highlighted
